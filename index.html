Install the apps to the home screen — both AquaTreat and PoolCheck install and run offline: open the app, then Android/Chrome → menu ⋮ → Install app; iOS/Safari → Share → Add to Home Screen.

Note: AquaTreat's batch report upload (parsing Excel lab sheets) needs the Python server (python app.py) and does not run on this static site — the dosing, chlorine field test, and single-sample ionic balance all work here.
